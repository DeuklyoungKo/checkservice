# Trend Scouter 🚀

**AI 기반 비즈니스 기회 포착 서비스**

Trend Scouter는 실시간 트렌드 데이터를 분석하여 1인 개발자나 소규모 팀이 수익화할 수 있는 유망한 비즈니스 아이디어를 제안합니다.

## ✨ Key Features
- **AI 트렌드 분석:** 글로벌 데이터를 기반으로 유망한 사업 기회 매일 업데이트
- **점수 시스템:** 실현 가능성, 수익 잠재력, 개발 난이도 수치화
- **심층 리포트:** 선택한 트렌드에 대한 시장 배경 및 구현 가이드 제공

## 🛠 Tech Stack & Design
- **Core:** Next.js 15 (App Router)
- **UI:** [shadcn/ui](https://ui.shadcn.com/) (New York Style)
- **Icons:** [Tabler Icons](https://tabler.io/icons)
- **Styling:** Tailwind CSS v4 ([Orange Theme](https://ui.shadcn.com/themes))
- **Typography:** [JetBrains Mono](https://www.jetbrains.com/lp/mono/) (font-mono 기반)
- **Support:** RTL (Right-to-Left) 지원

## 🚀 Getting Started

### Prerequisites
- Node.js 20+ 이상
- npm 또는 pnpm

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```
[http://localhost:3000](http://localhost:3000)에서 실시간으로 확인할 수 있습니다.

## 📁 Project Structure
- `src/app`: 페이지 및 레이아웃 설정
- `src/components/ui`: shadcn/ui 재사용 컴포넌트
- `src/lib`: 유틸리티 및 API 로직
- `supabase`: 데이터베이스 스키마 및 마이그레이션
